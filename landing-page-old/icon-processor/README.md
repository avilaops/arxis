**High-performance icon processor** written in Rust that generates optimized icons for all platforms.

## 🚀 Quick Start

```bash
# Build
cargo build --release

# Run
./target/release/arxis-icons --input ./input --output ./output
```

See [QUICKSTART.md](QUICKSTART.md) for full instructions.

## ✨ What It Does

Generates **38-44 optimized icons** from a single image for:
- 🌐 Web (favicon)
- 🍎 iOS/macOS
- 🤖 Android
- 🪟 Windows
- 📱 Social Media

Plus PWA manifest, HTML meta tags, and README!
