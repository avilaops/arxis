### Added
- Initial project structure
- Core DataFrame and Series implementation with Apache Arrow backend
- Scientific data types (Quaternions, Weyl Spinors, Geodesic coordinates)
- Expression system for lazy evaluation
- Basic operations (filter, group_by, select, with_column)
- I/O stubs for Parquet and CSV
- Comprehensive error handling
- Examples demonstrating basic usage and scientific types
- Benchmark suite with Criterion

- ⚛️ Quaternion support for 3D/4D rotations
- 🔬 Weyl Spinor support for particle physics

- 📊 DataFrame operations: filter, select, group_by
- 📈 Statistical functions: mean, std, sum
- 🎯 Expression builder with operator overloading
