## ⚡ Quick Start

\`\`\`bash
cargo add arxis_quaternions avila-math avila-telemetry
\`\`\`

\`\`\`rust
use avila_math::geometry::{Quat3D, Tesseract};
use avila_math::tensor::Tensor4D;
use std::f64::consts::PI;

// 4D Geometry
let tesseract = Tesseract::new();
println!("Vertices: {}", tesseract.vertices.len()); // 16

// Quaternion rotations
let q = Quat3D::from_axis_angle([0.0, 1.0, 0.0], PI / 2.0);
let rotated = q.rotate_vector([1.0, 0.0, 0.0]);

// 4D Tensors with Conv4D
let tensor = Tensor4D::new(10, 3, 32, 32); // batch, channels, height, width
\`\`\`

## 🎯 Features

- ✅ **4D Geometry**: Tesseract, 24-cell, Simplex, projections, SO(4) rotations
- ✅ **Tensors**: N-dimensional (0D-4D) with Conv4D layers
- ✅ **Quaternions**: 3D rotations, dual quaternions, rigid bodies
- ✅ **Physics**: Gravitational waves, General Relativity, LISA pipeline
- ✅ **Time Series**: ARIMA, anomaly detection, forecasting
- ✅ **Quantum Rendering**: Path integral photon transport
- ✅ **Big Data**: Polars, Arrow, Parquet integration
- ✅ **Machine Learning**: K-Means, PCA, Conv4D CNNs
- ✅ **NASA-Grade**: Tested for space missions
